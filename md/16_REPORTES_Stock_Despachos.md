# 16.05 STOCK POR DESPACHOS
### Módulo 16 — Reportes

> **Versión:** 1.0 — Marzo 2026
> **Audiencia:** Usuarios finales del sistema (sin conocimientos técnicos)
> **Pantalla en el sistema:** Menú → Reportes → Stock Despachos

---

## ¿Para qué sirve esta pantalla?

Muestra el stock desglosado por número de despacho de aduana. Permite controlar qué unidades de cada despacho están en qué depósito. Solo aplica a artículos sin número de serie (NSE).

## Cómo acceder

Desde el menú principal seleccioná:
**Reportes → Stock Despachos**

---

## Filtros

Permite filtrar la lista de artículos por código, tipo, grupo, marca o variante.

---

## Columnas de la grilla

| Columna | Descripción |
|---|---|
| Despacho | Número de despacho de aduana |
| Código | Código del artículo |
| Depósito | Nombre del depósito donde está el stock |
| Cantidad | Unidades de ese despacho en ese depósito |
| Artículo | Descripción del artículo |
| Tipo | Tipo de artículo |
| Grupo | Grupo |
| Marca | Marca |
| Proveedores | Proveedor |
| Variante | Variante |

> ℹ️ Esta pantalla no muestra artículos con número de serie (NSE). Para esos equipos, consultá el módulo de Logística.
