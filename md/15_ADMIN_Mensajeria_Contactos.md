# 15.25 MENSAJERÍA CONTACTOS
### Módulo 15 — Administración

> **Versión:** 1.0 — Marzo 2026
> **Audiencia:** Usuarios finales del sistema (sin conocimientos técnicos)
> **Pantalla en el sistema:** Menú → Administración → Accesos → Mensajería Contactos

---

## ¿Para qué sirve esta pantalla?

La pantalla de **Mensajería Contactos** permite gestionar los contactos vinculados a las plataformas de mensajería del sistema (WhatsApp, etc.). Cada contacto representa una cuenta de mensajería que puede asociarse a un usuario del sistema para recibir notificaciones y comunicaciones automáticas. Desde aquí podés:

- Ver todos los contactos de mensajería registrados en el sistema
- Asociar un contacto a un usuario del sistema
- Eliminar contactos que ya no se utilicen

---

## ¿Cómo acceder?

**Menú → Administración → Accesos → Mensajería Contactos**

---

## Tabla de contactos

La pantalla muestra una tabla con todos los contactos de mensajería registrados:

| Columna | Descripción |
|---|---|
| (Acción) | Botón para eliminar el contacto. |
| Plataforma | Plataforma de mensajería (ej: WhatsApp) con su logo identificador. |
| Nombre Completo | Nombre y apellido del contacto. |
| Estado | Estado actual del contacto en la plataforma. |
| Fecha inicio | Fecha en que se registró el contacto. |
| Usuario | Usuario del sistema asociado a este contacto. Se puede asociar/cambiar desde aquí. |

Los contactos marcados como **CONTACTO INTERNO** son del sistema y no se pueden reasignar.

---

## Acciones disponibles

| Acción | Descripción |
|---|---|
| **Eliminar** (ícono borrar) | Elimina el contacto de la plataforma de mensajería. |
| **Asociar usuario** | Selector que permite vincular el contacto a un usuario del sistema. Al asociar, el usuario recibirá notificaciones a través de ese contacto de mensajería. |

---

## Notas importantes

- Los contactos de mensajería son cuentas de plataformas externas (ej: número de WhatsApp) que el sistema utiliza para enviar notificaciones automáticas.
- Cada contacto debe estar asociado a un **usuario del sistema** para que las notificaciones lleguen a la persona correcta.
- Los contactos marcados como internos son administrados por el sistema y no pueden modificarse desde esta pantalla.
