# 6.16 ML MENSAJES
### Módulo 6 — Mercado Libre

> **Versión:** 1.0 — Marzo 2026
> **Audiencia:** Usuarios finales del sistema (sin conocimientos técnicos)
> **Pantalla en el sistema:** Ventana emergente accesible desde ML Ventas y ML Preparar Ítems

---

## ¿Para qué sirve esta pantalla?

**ML Mensajes** es una ventana emergente que muestra el historial de mensajes entre el vendedor y el comprador de una venta de MercadoLibre. Desde aquí podés:

- Leer todos los mensajes intercambiados en el hilo de esa venta
- Ver quién envió cada mensaje (vendedor o comprador) y cuándo
- Consultar el historial antes de contactar al comprador

> ℹ️ Esta pantalla es solo de **lectura**. No permite enviar mensajes nuevos desde Caddis. Para responder, accedé directamente al portal de MercadoLibre.

## Cómo acceder

Esta pantalla es una ventana emergente. Se accede desde:
- **ML Ventas** → seleccionar una venta → botón **Mensajes**
- **ML Preparar Ítems** → seleccionar una venta → botón **Mensaje**

---

## Descripción de la pantalla

La ventana muestra una lista de mensajes del hilo de comunicación de la venta, ordenados cronológicamente.

| Elemento | Descripción |
|---|---|
| **Remitente** | Nombre de quien envió el mensaje (vendedor o nombre del comprador) |
| **Fecha** | Fecha y hora del mensaje |
| **Texto** | Contenido del mensaje |

Los mensajes del vendedor y del comprador se muestran con colores alternados para facilitar la lectura del hilo.

---

## Cuándo usar esta pantalla

- Para revisar si el comprador informó algún problema con el envío antes de llamarlo.
- Para verificar si ya se respondió una consulta antes de volver a contactar.
- Para tener contexto de la comunicación previa al gestionar un reclamo.
