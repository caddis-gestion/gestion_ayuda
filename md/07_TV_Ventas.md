# 7.11 TV VENTAS
### Módulo 7 — Tiendas Virtuales

> **Versión:** 1.0 — Marzo 2026
> **Audiencia:** Usuarios finales del sistema (sin conocimientos técnicos)
> **Pantalla en el sistema:** Menú → Tiendas Virtuales → TV Ventas

---

## ¿Para qué sirve esta pantalla?

**TV Ventas** es el panel de seguimiento de todas las ventas de las tiendas virtuales. Desde aquí podés:

- Consultar y filtrar las ventas por estado, fecha, envío y facturación
- Ver el detalle completo de cada venta
- Imprimir etiquetas de envío
- Actualizar ventas manualmente, emitir remitos y generar lotes de etiquetas

## Cómo acceder

Desde el menú principal seleccioná:
**Tiendas Virtuales → TV Ventas**

---

## Panel izquierdo: Filtros

| Campo | Descripción |
|---|---|
| **TIENDA** | Selector de tienda virtual (banner). |
| **USUARIO** | Cuenta de la tienda. |
| **DEPOSITO** | Filtrar por depósito/punto de venta Caddis. |
| **DESDE / HASTA** | Rango de fechas. |
| **BUSQUEDA** | Texto libre: ID de venta, número de orden o número de remito (ej: R-00001-00001234). |
| **ESTADO** | Estado de la venta según la tienda (opciones según plataforma). |
| **ENVIO** | Tipo de envío. |
| **FACTURADAS** | Filtrar por ventas facturadas (SI), no facturadas (NO) o todas. |
| **VER POR** | Ordenar por fecha de CREACION o MODIFICACION. |

### Sección OPERACIONES MASIVAS

| Botón | Descripción |
|---|---|
| **Actualización manual de ventas** | Descarga las ventas nuevas desde la tienda virtual. |
| **Emisión manual de remitos** | Genera remitos pendientes para ventas sin remito emitido. |
| **Imprimir lote de etiquetas** | Imprime las etiquetas de envío para todas las ventas filtradas. |
